# AppEcology
## 项目介绍
将一个个好玩，有趣，实用的功能做成模型，在网站上直接使用。形成一个工具箱类型的网站。
## 技术栈

| 技术栈        | 描述        | 官网                  |
|------------|-----------|---------------------|
| odoo       | python web框架 用来部署模块 | https://odoo.com/   |
| python     | 开发大多使用python语言 | https://python.org/ |
| javascript | 少部分使用     |  |


## 已有功能
1. 运动刷步数（微信，支付宝）
2. 图床
3. 每日更新面板
4. 自动化（简陋版）
